# ⚔️ HabitHero — Геймифицированный RPG-трекер привычек

Одностраничное приложение (SPA) для отслеживания привычек в стиле RPG, созданное на **HTML + CSS + Vanilla JS**. Без фреймворков и инструментов сборки.

---

## 🗂 Архитектура

```text
habit-hero/
│
├── index.html                  # Оболочка приложения + упорядоченные теги script
│
├── css/
│   ├── variables.css           # Дизайн-токены (цвета, отступы, типографика)
│   ├── base.css                # Сброс стилей, body, типографика, скроллбар
│   ├── layout.css              # Экраны, навигация, сетки страниц, адаптив
│   ├── components.css          # Стили всех UI-компонентов
│   └── animations.css          # Keyframes + утилитарные классы анимаций
│
└── js/
    ├── data/                   # Статические, замороженные данные (без логики)
    │   ├── habits.js           # HABITS_DATA — 4 ежедневных квеста
    │   ├── achievements.js     # ACHIEVEMENTS_DATA — 9 достижений + логика check()
    │   └── shopItems.js        # SHOP_ITEMS_DATA — 4 товара в магазине
    │
    ├── models/
    │   └── Character.js        # Класс Character — чистая модель данных, сериализация
    │
    ├── services/               # Чистая бизнес-логика без состояния (без работы с DOM)
    │   ├── StorageService.js   # Абстракция над localStorage
    │   ├── XPService.js        # Расчет опыта (XP) и уровней
    │   ├── StreakService.js    # Управление ежедневными сериями (стриками)
    │   └── AchievementService.js # Оценка выполнения достижений
    │
    ├── store/
    │   └── Store.js            # Реактивное хранилище состояния (паттерн Observer)
    │
    ├── components/             # UI-слой — HTML-шаблоны + события DOM
    │   ├── Component.js        # Абстрактный базовый класс (render/template/bindEvents)
    │   ├── Toast.js            # Синглтон для управления всплывающими уведомлениями
    │   ├── Confetti.js         # Эффект конфетти при повышении уровня
    │   ├── Modal.js            # Очередь модальных окон (уровни, достижения)
    │   ├── Navigation.js       # Навигация по вкладкам, генерирует nav:change
    │   ├── ProfileCard.js      # Отображение характеристик героя
    │   ├── StreakBanner.js     # Панель серий с контрольными точками
    │   ├── QuestBoard.js       # Сетка квестов, генерирует quest:completed
    │   ├── AchievementsPage.js # Сетка достижений
    │   ├── ShopPage.js         # Сетка магазина, генерирует shop:purchase
    │   └── CharacterCreation.js # Экран создания персонажа, генерирует character:created
    │
    └── app.js                  # Класс App — корневой контроллер, связывающий всё воедино
```

---


## 🎮 Возможности

- **Создание персонажа** — имя + 4 класса аватаров.
- **Ежедневные квесты** — 4 привычки с наградами в виде опыта и золота, сбрасываются каждый день.
- **Система уровней** — 100 XP на уровень, модальное окно повышения уровня + конфетти.
- **Отслеживание серий** — счетчик дней подряд с вехами на 3/7/30 дней.
- **9 достижений** — включая одно секретное.
- **Магазин** — премиальные аватары, темная тема, бусты опыта.
- **Полное сохранение прогресса** — все данные хранятся в `localStorage`.
```
